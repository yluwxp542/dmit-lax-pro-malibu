## 适合哪些人，不适合哪些人

先说适合的场景，我自己用下来觉得这类需求最契合：

1. **搭代理/翻墙节点**：CN2 GIA 线路对这类用途帮助最直观——晚高峰稳定性远好于普通国际线路，1Gbps 带宽对单节点个人使用完全够。
2. **个人博客/小网站**：国内访客打开速度快，1核1G跑 WordPress 或者静态站没问题。
3. **流媒体解锁**：DMIT 全系配原生 IP，实测对 Netflix、TikTok 之类的流媒体解锁能力不错（官方不作保证，以实测为准）。
4. **外贸轻量业务**：需要一个稳定的美西节点收发邮件、连接 Google Workspace 之类的，这个配置完全够用。

不适合的情况同样要说清楚：

- 要跑数据库密集型应用（MySQL 重负载）→ 1GB 内存不够
- 需要跑多个站、高并发 → 看 PalmSpring 或更高
- 流量需求超过 1TB/月 → 上 PalmSpring，或者换看 LAX T1 系列（超量限速更高）
- 对成本极度敏感、线路质量无所谓 → WEE 或者 EB 系列更划算
